Flexbox

Noțiuni de bază

Flexible Box Layout (Flexbox) – este un un modul CSS care definește un set de proprietăți pentru poziționarea, alinierea și distribuirea spațiului între elementele dintr-un container, chiar dacă dimensiunea acestora este necunoscută sau dinamică.

Ideea de bază este de a separa elementele într-un container bloc (flex-container) și copiii lui (flex-elemente).

Containerul-flexbox poate modifica lățimea și înălțimea copiilor săi, precum și direcția de aranjare a acestora, alinierea într-o coloană sau linie, ordinea în care sunt afișate elementele și distanța dintre ele. Un container extinde elementele pentru a se potrivi cel mai bine în spațiul disponibil sau le micșorează pentru a preveni supraaglomerarea.

Atentie...!!!

Într-un container-flex, majoritatea regulilor de aranjare a elementelor în funcție de tipul și fluxul documentelor sunt anulate.

  • Elementele își pierd tiparul, deci încetează să mai fie de tip inline sau bloc.
  • Elementele de tip bloc nu mai merg vertical unul sub celălalt.
  • Marginile de la legătura cu granița părintelui nu ies în exterior.
  • Marginile verticale ale elementelor nu se prăbușesc.
  • Funcționează marginile verticale automate.

  • Terminologie

    Ordinea normală a elementelor este determinată de fluxul documentului și de tipul acestora (bloc sau inline). În modelul Flexbox, direcția în care sunt așezate elementele este determinată de axele de direcție ale containerului de-a lungul căruia se aliniază elementele.

    • main axis – axa principală a containerului-flex, de-a lungul căreia sunt situate elementele. Direcția nu trebuie să fie orizontală, ea este controlată de proprietatea flex-direction.
    • main-start și main-end – elementele dintr-un container sunt întotdeauna situate de la main-start (începutul axei principale) până la main-end (sfârșitul axei principale).
    • cross axis – axa transversală, care este întotdeauna perpendiculară pe axa principală. Direcția sa depinde de axa principală și nu este setată în mod explicit.
    • cross-start și cross-end – începutul și sfârșitul axei transversale de-a lungul căreia sunt situate rândurile de elemente.

Proprietățile unui container

Un container-flex (denumit în continuare container) este părintele unui grup de elemente care trebuie reordonate. Proprietățile containerului sunt cele care controlează direcția axelor, multiliniaritatea și poziționarea elementelor în linie.

  • Proprietatea display

    Aceasta creează un container-flex, de tip bloc sau inline-block, în funcție de valoarea dată și setează contextul-flex pentru toți copiii (nu descendenții) containerului, transformându-i în elemente-flex.

    display: flex | inline-flex

    Exemplu

    Să creăm un meniu orizontal transformând ul.menu într-un container-flex. Apoi copiii săi, elementele li.item, vor fi amplasate de-a lungul axei principale, care în mod implicit sunt aranjate de la stânga la dreapta.

    • Home page
    • Blog
    • Podcasts
    • Contacts
    HTML

    <ul class="menu">
    <li class="item"><a href="" class="link">Home page</a></li>
    <li class="item"><a href="" class="link">Blog</a></li>
    <li class="item"><a href="" class="link">Podcasts</a></li>
    <li class="item"><a href="" class="link">Contacts</a></li>
    </ul>

    .menu {
    display: flex;
    }

    Containerul flex ul.menu nu influențează în niciun fel a.link în interiorul elementelor de listă, deoarece acestea sunt descendenții săi, nu copii. Copiii sunt cei care devin elemente-flex, în cazul nostru, elementele li.item.

    CSS

    * {
    box-sizing: border-box;
    }

    body {
    font-family: sans-serif;
    background-color: #f9f9fd;
    }

    .menu {
    padding: 10px;
    margin: 0;
    list-style: none;
    border-radius: 4px;
    background-color: #2196f3;
    box-shadow: 0px 2px 1px -1px rgba(0, 0, 0, 0.2),
    0px 1px 1px 0px rgba(0, 0, 0, 0.14), 0px 1px 3px 0px rgba(0, 0, 0, 0.12);

    /* Making ul.menu a Flex container */
    display: flex;
    }

    .menu .link {
    display: block;
    padding: 10px;
    border-radius: 4px;
    text-decoration: none;
    color: black;
    background-color: #fff;
    }

    .menu .link:hover {
    text-decoration: underline;
    background-color: olive;
    }
